During my internship at **AWS**, I had the opportunity to study cloud computing technologies and participate in the AWS Workforce Bootcamp while developing a real-world web game project based on AWS services.

Throughout the internship, I completed a series of hands-on laboratories covering networking, storage, backup, monitoring, and security. I also applied these technologies to develop a serverless web application by integrating **Amazon Cognito**, **Amazon DynamoDB**, **AWS Lambda**, **Amazon API Gateway**, **Amazon SQS**, **Amazon CloudWatch**, **AWS X-Ray**, and **AWS Amplify**.

Besides learning AWS services, I also implemented core game features such as real-time communication using **WebSocket**, an Elo-based matchmaking system, user authentication, and backend APIs. These experiences helped me improve not only my technical knowledge but also my ability to research new technologies, analyze problems, and deploy cloud-based applications.

During the internship, I always tried to complete assigned tasks on schedule, actively discussed technical issues with mentors and teammates, and continuously improved my project based on feedback.
